# Repository Overview

Website for projects, learning and portfolio.

## Setup Requirement

- Ruby and bundler

### Setup Steps

- Create new jekyll site and install necessary gem using bundle.

<pre> $ jekyll new --skip-bundle . 
 $ bundle install
</pre>

- Modify _config.yml for site



### Shortcuts

For Create a Note section, use ### Note

### CSS Design Specific

Bootstrap 5 class for table
Paste the following after table.

    {: .table .table-striped .table-hover }
